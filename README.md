# Juego de Sudoku 🧩

**Juego de Sudoku** es una herramienta interactiva diseñada tanto para jugar partidas manuales como para resolver tableros complejos de forma automática. El proyecto destaca por el uso de algoritmos de búsqueda para encontrar soluciones instantáneas.

## ✨ Funcionalidades Principales
* **Modo Interactivo:** Cuadrícula de 9x9 con validación de entrada (solo números del 1 al 9) para una experiencia de juego limpia.
* **Resolución Automática (Auto-Solve):** Motor de resolución basado en el algoritmo de **Backtracking** que completa cualquier tablero válido al instante.
* **Validador de Progreso:** Botón para verificar si los números ingresados por el usuario son correctos, resaltando errores en tiempo real.
* **Gestión de Tablero:** Opciones para limpiar la cuadrícula, generar un nuevo juego inicial o solucionar el actual.
* **Diseño Visual:** Interfaz amigable con colores diferenciados para distinguir los bloques de 3x3 y mejorar la legibilidad.

## 🛠 Tecnologías Utilizadas
* **Python:** Lenguaje base para la lógica del juego y el motor de resolución.
* **Tkinter:** Librería para el desarrollo de la interfaz gráfica de usuario (GUI).
* **Algoritmo de Backtracking:** Implementación recursiva para la búsqueda eficiente de soluciones lógicas.

## 🚀 Instalación y Ejecución

Asegúrate de tener instalado Python en tu computadora y sigue estos pasos:


# 1. Clona este repositorio
```bash
git clone [https://github.com/ClutchDev23/sudoku-solver.git](https://github.com/ClutchDev23/sudoku-solver.git)
```
# 2. Accede a la carpeta del proyecto
```bash
cd sudoku-solver
```
# 3. Inicia el juego
```bash
python sudoku.py
```
